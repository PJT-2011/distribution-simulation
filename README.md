# Distribution Simulation using Python

## Project Description
This project demonstrates five basic probability distributions using Python:

- Normal Distribution
- Uniform Distribution
- Exponential Distribution
- Binomial Distribution
- Poisson Distribution

Each distribution is simulated using NumPy and visualized using Matplotlib.

---

## How to Run

### 1. Install Python
Make sure Python is installed on your system.

### 2. Install Dependencies
Open terminal and run:
```bash
pip install numpy matplotlib
```

### 3. Run the Program
```bash
python main.py
```
